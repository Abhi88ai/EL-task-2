# EL-task-2

 Console-Based To-Do List Application

Project Overview
This is a simple **console-based To-Do List application** built using **Python**.  
The application allows users to manage daily tasks through a command-line interface and ensures **data persistence** using a text file.



## Objective
To implement a basic to-do list manager that:
Runs in the terminal
Supports adding, viewing, and removing tasks
Saves tasks permanently using file handling
